# API ENDPOINTS

## HTML API

### Root
- GET /

## JSON API

### Users
- GET /api/users
- GET /api/users/:id
- POST /api/users
- PATCH /api/users/:id

### Session
- GET /api/session
- DELETE /api/session

### Tracks
- GET /api/tracks
- GET /api/tracks/:id
- POST /api/tracks
- PATCH /api/tracks/:id
- DELETE /api/tracks/:id

### Comments
- GET /api/tracks/comments
- POST /api/tracks/comments
- DELETE /api/tracks/comments/:id
